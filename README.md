# CalmoraOS

CalmoraOS is a new x86-based operating system built from scratch.
It features a custom kernel and uses GRUB as its bootloader.

## About

CalmoraOS is a hobby operating system project focused on learning
and developing a custom operating system from the ground up.

## Features

- x86 architecture support
- Custom kernel
- GRUB bootloader
- Assembly and C based development
- Basic system components

## Status

Current version: Alpha 0.1

CalmoraOS is currently in early development.
Some features may not work correctly.

## Running CalmoraOS

You can test CalmoraOS using QEMU.

Example:

qemu-system-i386 -cdrom CalmoraOS.iso

## Reporting Bugs

Found a problem?

Please create a new Issue in this GitHub repository.

When reporting a bug, include:

- CalmoraOS version
- How you started the system
- Error message or screenshot
- Steps to reproduce the problem
